Paso a paso

Como instalar MLA POS en Windows

El proceso tarda menos de dos minutos. Sigue estos pasos en orden.

  1. 1

    Descargá el instalador

    Tocá el botón Descargar para Windows arriba. Se bajará el archivo MLA-POS-Setup.exe desde GitHub.

  2. 2

    Abre el archivo descargado

    Anda a la carpeta Descargas y hace doble clic en el instalador. Si Windows pregunta, elige permitir la ejecución.

  3. 3

    Si aparece “Windows protegió tu PC”

    Es normal en apps sin firma comercial. Clic en Más información y después en Ejecutar de todas formas. No significa que la app tenga virus.

    Windows protegió tu PC

    Editor desconocido

    Más información Ejecutar de todas formas
  4. 4

    Completa el asistente

    Siguiente → elige carpeta (o deja la predeterminada) → Instalar. Al finalizar, marca abrir la app si quieres probarla al instante.

  5. 5

    Listo: abre la app MLA POS

    Busca el acceso directo en el escritorio o en el menú Inicio. La primera vez puede tardar unos segundos en cargar la base local.

¿Por qué Windows la marca como no confiable?

Editor desconocido ≠ app peligrosa

Qué te muestra Windows

SmartScreen avisa cuando el instalador no tiene certificado de firma de código emitido por un editor reconocido (Microsoft, Adobe, etc.). Por eso dice “editor desconocido” o que la app “no es confiable”.

Por qué pasa con MLA POS

MLA POS se distribuye directo desde GitHub, sin pasar por una tienda oficial. Los certificados de firma cuestan cientos de dólares al año; muchas apps independientes —como esta— se publican sin firmar hasta tener ese presupuesto.

¿Es seguro instalarla?

Sí, siempre que descargues desde este sitio o desde el release oficial en GitHub Releases. El código fuente y cada versión quedan públicos en el repositorio. Windows solo no puede verificar automáticamente quién la publicó; vos confirmás la instalación con “Ejecutar de todas formas”.

  • Descargá solo desde el botón de esta página o GitHub Releases.
  • No uses instaladores que te manden por WhatsApp o enlaces raros.
  • En el futuro podemos agregar firma de código para quitar el aviso.
